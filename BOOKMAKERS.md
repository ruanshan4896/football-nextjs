# Danh sách Bookmaker ID phổ biến

Dựa trên API-Football, một số bookmaker phổ biến:

## Bookmaker IDs:
- **8** = Bet365 (hiện tại đang dùng)
- **5** = Bwin
- **6** = Betfair
- **3** = Pinnacle
- **1** = William Hill
- **2** = Ladbrokes

## Lưu ý:
- Để lấy danh sách đầy đủ, gọi API: `GET /odds/bookmakers`
- SBO/SBOBET có thể có ID khác, cần kiểm tra từ API
- Không phải bookmaker nào cũng có đầy đủ kèo cho mọi giải đấu
